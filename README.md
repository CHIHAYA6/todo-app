# Todoアプリ

## 概要

JavaScriptを用いて作成したタスク管理アプリです。

タスクの追加・削除・完了管理を行うことができます。

（GitHub PagesのURLを記載）

## 使用技術

* HTML
* CSS
* JavaScript

## 実装機能

* タスク追加
* タスク削除
* 完了状態の切り替え
* LocalStorageによるデータ保存

## 工夫した点

* ページを再読み込みしてもタスクが消えないようにLocalStorageを利用しました。
* シンプルで使いやすいデザインを意識しました。
